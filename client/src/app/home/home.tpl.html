angular-bootstrap-kickstart is an opinionated kickstart for single page application development with AngularJS.

Step 1 clone repository from github

$ git clone https://github.com/yahoohung/angular-bootstrap-kickstart.git

Step 2 install bower, npm and dependencies.

$ npm install gulp bower -g && npm install && bower install

Step 3 run it in development mode

$ gulp serve

Deploy build a production release

$ gulp serve:dist
